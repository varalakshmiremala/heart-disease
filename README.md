# ❤️ Heart Disease Prediction AI Agent

## Project Overview

This project is an Intelligent Healthcare Monitoring Agent that predicts the risk of heart disease using a Machine Learning model.

The ML model was trained in Google Colab and deployed as an AI Agent using Streamlit.

## Features

- Heart Disease Prediction
- Patient Health Analysis
- Health Recommendations
- Patient Monitoring Logs
- Interactive Streamlit Interface

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## Files

- app.py
- agent.py
- heart_model.pkl
- scaler.pkl
- requirements.txt

## Run the Project

```bash
pip install -r requirements.txt
python -m streamlit run app.py
```
